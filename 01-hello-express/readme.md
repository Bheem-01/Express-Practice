/*
=========================================
PROGRAM EXPLANATION
=========================================

1. Import the Express framework using require().
   This gives access to all Express features.

2. Create an Express application by calling
   express(). The returned object is stored
   in the variable 'app'.

3. Define the port number (3000). The server
   will listen for client requests on this port.

4. Start the server using app.listen().

5. Once the server starts successfully, the
   callback function executes and prints
   confirmation messages in the terminal.

6. After starting, the server keeps running
   and waits for incoming HTTP requests.

=========================================
EXECUTION FLOW
=========================================

Run Program (node app.js)
        ↓
Load Express
        ↓
Create Express App
        ↓
Assign Port Number (3000)
        ↓
Start Server (app.listen)
        ↓
Server Starts Successfully
        ↓
Print Success Messages
        ↓
Wait for Client Requests
*/
